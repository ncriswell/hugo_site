Overview

This site will serve as a repository for analysis of the 2019 The League Season Thunderdome: The Sequel NFL Fantasy season. Our league is hosted on the fantastic Sleeper platform and data are is are (ok, fine) is pulled using the Sleeper API. The analysis is completed using the R programming environment and pushed out as a website via the blogdown package, utilizing the Hugo static web framework. Netlify is the hosting platform, and all code can be found on my GitHub page.

I’ll try to make weekly updates to the site as the season goes along and will also be doing write-ups here and there that contain analyses that span multiple seasons, go back to a particular season, or are just something which might be interesting to look at. Go back to the home page to and look for “TLSTTS” in the post title for those ones. These posts will obviously focus on analyzing the league results through data; if you want analysis that focuses on analyzing the league through the lens of some kind of constantly shifting mythology involving a pantheon of several gods aped from various world traditions, see the weekly newsletter that Joe sends out.

Round Robin Analysis

The prior graph shows weekly performance against the median and average. This brings up a question about how each owner performs against all other owners each week-not just how they do against the person they are playing. We can create a graph that will show for each week how many teams each owner would have beat if we had to play every other owner every week. This view is a handy way to find owners who are consistently benefiting/suffering from favorable/unfavorable matchups. Each “row” is a week. Colors are added to show the win/loss outcome, and the number of round robin wins is indicated with black text.

This graph does a really good job of showing people who are benefiting from favorable matchupes (Max) and people who might have gotten robbed here and there (Zach, James). Joe, not sure if you want to use this for your weekly rankings, but if you do, you might find yourself slipping a bit…
